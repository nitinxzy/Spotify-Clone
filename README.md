# 🎧 Spotify Clone (HTML & CSS)

A simple front-end clone of the popular music streaming platform **Spotify**, built using pure HTML and CSS.
This project focuses on replicating the UI design and layout to improve front-end development skills.

---

## 🚀 Features

* 🎵 Modern and clean Spotify-inspired user interface
* 📱 Responsive design (mobile + desktop friendly)
* 🎨 Styled using pure CSS (no frameworks)
* 🧩 Structured layout with sidebar, navbar, and music sections

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## 📂 Project Structure

```
spotify-clone/
│── index.html
│── style.css
│── images/
│── assets/
```

---

## 💡 Purpose of the Project

This project was created to practice front-end development concepts such as:

* Layout design
* Flexbox & positioning
* UI structuring
* Styling techniques
